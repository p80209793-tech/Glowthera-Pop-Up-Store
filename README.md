# Bodyand Pop-up Store Landing Page

This is the source code for the Bodyand Pop-up Store landing page.

## Project Structure

- `index.html`: Main HTML file.
- `style.css`: Main stylesheet.
- `assets/`: Directory containing all image assets.

## How to Run

1. Clone or download this repository.
2. Open `index.html` in your web browser.

## Deployment

This project is static and can be deployed on GitHub Pages, Netlify, or Vercel by simply uploading this folder.
